Manual transfer from Google Drive to GitHub ( Can also down

Select Google Drive from the task manager

Click on preferences

Then Click on gear again and go down to the Google Drive streaming location section. Provide the location where you wish to stream the Drive files. After doing so, whatever changes are made in the My drive folder of Google Drive are reflected in the local folder.

Now copy paste your files from the local drive folder to Git repo folder and commit the changes and push changes to Github as Required

Note that streaming files only saves a version in local and is available when online. If you wish to access the Drive folder offline one can use the mirroring option.
